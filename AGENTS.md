# Agents

- Always use `uv` commands instead of `pip` commands.
- Use `ty` (https://docs.astral.sh/ty/) type checking. Run `uv run ty check` before committing.
- Use `typer` (https://typer.tiangolo.com/) for CLI argument parsing.
- Use `ruff` for linting and formatting. Run `uv run ruff check` and `uv run ruff format` before committing.
